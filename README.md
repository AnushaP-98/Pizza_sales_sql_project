## Pizza Sales SQL & Excel Project

## Project Overview
This project analyzes pizza sales data using SQL for data exploration and Excel for dashboard visualization.

## Tools Used
- SQL (SQL Server)
- Microsoft Excel
- Pivot Tables

## Dataset
- pizza_sales_raw.csv

## Key Analysis Performed
- Total Revenue
- Best & Worst Selling Pizzas
- Sales by Category
- Sales by Size
- Monthly & Daily Trends

## Dashboard
- ![Dashboard](dashboard_preview.jpg)

## Insights
- The Classic category generated highest revenue
- Large size pizzas contributed most sales
- Friday had the highest sales

## Conclusion
SQL helped extract insights efficiently, and Excel was used to build an interactive dashboard.
